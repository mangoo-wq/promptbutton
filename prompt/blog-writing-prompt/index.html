블로그 글쓰기 AI

블로그 글쓰기 AI를 찾는 사람은 대개 빈 화면에서 막힙니다. 이 페이지는 제목, 소제목, 초안을 한 번에 잡아서 시작 장벽을 낮추는 데 집중합니다.

이럴 때 쓰세요
  • 블로그 초안을 빨리 만들고 싶을 때
  • 제목과 소제목 구성이 필요할 때
  • 정보성 글의 뼈대를 잡고 싶을 때

정중한 버전

예시 입력 / 출력

어떻게 넣고 어떤 결과를 기대하면 되는지 한 번에 보이게 구성했습니다.

예시 입력
주제: 직장인을 위한 회의록 정리 팁 / 대상: 직장인 / 톤: 친절하고 실용적
예시 출력
제목 예시: 회의록 정리, 10분 안에 끝내는 법 / 소제목: 왜 회의록이 항상 늦어질까, 빨리 정리하는 순서, AI를 활용하는 방법 / 본문: 회의 직후 핵심 논의와 액션 아이템부터 정리하면 시간을 줄일 수 있다...
