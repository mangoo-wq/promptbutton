긴 글 요약 AI

긴 글 요약 AI를 찾는 사람은 시간이 없거나 핵심만 빨리 파악하고 싶습니다. 이 페이지는 “짧게, 핵심만, 바로 써먹을 수 있게” 요약하는 데 맞춰져 있습니다.

이럴 때 쓰세요
  • 긴 보고서를 빨리 읽어야 할 때
  • 뉴스 기사 핵심만 파악하고 싶을 때
  • 공부용 요약이 필요할 때

바로 복붙하는 프롬프트

기본/짧은/정중한 버전을 같이 두어 상황에 맞게 바로 써볼 수 있게 했습니다.

기본 버전
아래 글을 읽고 1) 한 줄 요약 2) 핵심 내용 3개 3) 꼭 기억할 문장 1개 4) 실무/학습에 바로 쓸 포인트 순서로 정리해줘. 글: {{text}}
짧은 버전
아래 글을 핵심만 3줄로 요약해줘. 글: {{text}}
정중한 버전
아래 글을 초보자도 이해하기 쉽게 요약해줘. 어려운 말은 쉬운 표현으로 바꿔줘. 글: {{text}}

예시 입력 / 출력

어떻게 넣고 어떤 결과를 기대하면 되는지 한 번에 보이게 구성했습니다.

예시 입력
긴 기사 또는 보고서 본문
예시 출력
한 줄 요약: 생성형 AI 활용이 업무 생산성에 미치는 영향이 커지고 있다. / 핵심 내용: 반복 업무 자동화, 초안 작성 속도 향상, 검토 필요성 증가 / 기억할 문장: AI는 속도를 높이지만 검토를 대신하진 않는다.

자주 묻는 질문

이 페이지에 실제로 같이 따라붙는 고민들을 정리했습니다.

  • 논문 요약에도 쓸 수 있나요?
    가능합니다. 다만 전문 용어가 많은 경우 “쉽게 풀어서 설명해줘” 조건을 넣는 게 좋습니다.
  • 요약 길이를 조절할 수 있나요?
    네. 3줄, 5문장, bullet 5개처럼 원하는 형식을 함께 적으면 됩니다.
