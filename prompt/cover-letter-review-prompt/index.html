2층: 바로 복붙하는 프롬프트

검색으로 들어온 사람이 바로 써볼 수 있도록 기본/짧은/정중한 버전을 함께 둡니다.

기본 버전
아래 자기소개서를 읽고 1) 어색한 표현 2) 너무 추상적인 문장 3) 근거가 부족한 부분을 찾아 수정해줘. 문체는 자연스럽고 과장되지 않게, 지원 직무에 맞는 설득력이 느껴지게 다듬어줘. 자기소개서: {{cover_letter}} 직무: {{job_role}}
짧은 버전
아래 자소서를 자연스럽게 첨삭해줘. 과장된 표현은 줄이고, 설득력 있게 수정해줘. 자소서: {{cover_letter}}
정중한 버전
아래 자기소개서를 채용 담당자가 읽기 쉽게 다듬어줘. 문장 흐름, 표현, 설득력을 개선하고 수정 이유도 짧게 알려줘. 자소서: {{cover_letter}} 직무: {{job_role}}

예시 입력 / 출력

어떻게 넣고 어떤 결과를 기대하면 되는지 한 번에 보이게 구성했습니다.

예시 입력
저는 항상 성실함을 바탕으로 맡은 일을 책임감 있게 수행해왔습니다. 특히 대학교 팀프로젝트에서 리더를 맡아 팀원들을 이끌며 좋은 성과를 만들었습니다.
예시 출력
저는 맡은 일을 끝까지 책임지는 태도를 중요하게 생각합니다. 대학 팀프로젝트에서는 리더를 맡아 역할을 조율하고 일정 관리를 주도했고, 그 결과 프로젝트를 기한 내에 마무리하며 좋은 평가를 받았습니다.
