회의록 정리 AI

회의록 정리 AI를 찾는 사람은 보통 “말이 많고 시간이 없고, 액션 아이템만 뽑고 싶다”는 문제를 안고 들어옵니다. 이 페이지는 그 문제를 가장 빠르게 해결하는 프롬프트부터 보여줍니다.

이럴 때 쓰세요
  • 회의 내용을 빠르게 정리할 때
  • 발언자별 핵심 내용이 필요할 때
  • 액션 아이템만 뽑고 싶을 때

바로 복붙하는 프롬프트

기본/짧은/정중한 버전을 같이 두어 상황에 맞게 바로 써볼 수 있게 했습니다.

기본 버전
아래 회의 내용을 읽고 회의록으로 정리해줘. 1) 회의 목적 2) 핵심 논의 내용 3) 결정 사항 4) 담당자별 액션 아이템 5) 다음 일정 순서로 정리하고, 표 대신 읽기 쉬운 bullet 형식으로 써줘. 회의 내용: {{meeting_text}}
짧은 버전
아래 내용을 회의록으로 짧게 정리해줘. 결정 사항, 액션 아이템, 다음 일정만 보여줘. 내용: {{meeting_text}}
정중한 버전
아래 회의 내용을 팀에 공유할 수 있는 깔끔한 회의록 형태로 정리해줘. 회의 목적, 핵심 논의, 결정 사항, 후속 액션, 일정까지 포함해줘. 내용: {{meeting_text}}

예시 입력 / 출력

어떻게 넣고 어떤 결과를 기대하면 되는지 한 번에 보이게 구성했습니다.

예시 입력
신규 랜딩페이지 오픈 일정을 논의했다. 디자이너는 수요일까지 시안을 완료하고, 개발자는 금요일까지 반영하기로 했다. 마케팅팀은 다음 주 월요일부터 광고를 시작한다.
예시 출력
- 회의 목적: 신규 랜딩페이지 오픈 일정 조율 - 핵심 논의: 디자인 일정, 개발 반영 일정, 광고 시작 시점 확인 - 결정 사항: 수요일 시안 완료, 금요일 개발 반영 완료, 다음 주 월요일 광고 시작 - 액션 아이템: 디자이너-시안 완료 / 개발자-반영 / 마케팅팀-광고 집행 - 다음 일정: 다음 주 월요일 광고 시작
