빠른 사용법

아래 3단계만 따라 하면 바로 실무에 쓸 수 있습니다.

Step 1

상황 고르기

아래 실전 예시나 기본 프롬프트 중 지금 상황과 가장 가까운 템플릿을 고릅니다.

Step 2

내용 넣고 자동 교체

아래 입력칸(빈칸)만 채우면 {{변수명}}이 실제 내용으로 바뀐 상태로 복사됩니다.

Step 3

AI로 바로 실행

상단 바로가기로 실행한 뒤, 결과가 약하면 품질용/복구용 프롬프트로 재요청합니다.

바로 복붙하는 프롬프트

헷갈리면 1번부터 순서대로 쓰세요. 대부분 1→2만으로 바로 실무 사용 가능합니다.

1번

기본안

일단 빠르게 초안을 뽑아야 할 때. 첫 실행은 이걸로 시작하세요.

2번

정확도 점검

누락/모호함/우선순위를 더 깔끔하게 정리하고 싶을 때 사용합니다.

3번

다시쓰기

말투나 구조가 아쉬울 때, 결과를 자연스럽게 재정리할 때 씁니다.

복사 후 바로 실행

프롬프트를 복사한 뒤 원하는 AI를 바로 열어 테스트하세요.

ChatGPT 열기Claude 열기Gemini 열기
1) 기본안 (먼저 실행)
아래 회의 내용을 읽고 팀 공유용 회의록 초안을 만들어줘. 형식은 bullet. 순서: 1) 회의 목적 2) 핵심 논의 3) 결정 사항 4) 담당자별 액션 아이템(담당자/마감일/완료조건) 5) 다음 일정. 회의 내용: {{meeting_text}}
2) 정확도 점검
아래 회의 내용을 바탕으로 회의록을 품질검수 모드로 작성해줘. 규칙: 1) 사실/추정 구분 2) 빠진 정보는 [확인 필요] 표시 3) 각 액션 아이템에 담당자·마감일·우선순위 포함 4) 마지막에 리스크와 후속 확인 질문 3개 추가. 내용: {{meeting_text}}
3) 다시쓰기
직전 회의록 결과가 아쉬웠어. 같은 내용을 다시 정리해줘. 조건: 1) 중복 제거 2) 핵심 결정 3줄 요약 3) 액션 아이템만 별도 묶음 4) 팀 공지에 바로 붙여넣는 톤. 회의 내용: {{meeting_text}}

내용만 넣고 자동 교체 복사

여기서 변수는 빈칸입니다. 예: {{meeting_text}} → 회의 메모. 아래 칸을 채우면 자동으로 바뀐 상태로 복사됩니다.

빈칸을 안 채운 항목은 원래 표시({{변수명}}) 그대로 남습니다.

실전 예시 7개

상황별 프리셋입니다. 그대로 복사하거나, 위 입력칸(빈칸)을 채운 뒤 ‘변수 채워서 복사’를 누르면 바로 실행할 수 있습니다.

예시 1. 주간 팀 운영회의
아래 주간 팀 운영회의 메모를 팀 공유용 회의록으로 정리해줘. 형식: 1) 핵심 요약 2) 결정사항 3) 담당자/마감일 포함 액션 아이템 4) 블로커 5) 다음 회의 안건. 메모: {{meeting_text}}
예시 2. 클라이언트 미팅 결과 공유
아래 클라이언트 미팅 메모를 내부 보고용 회의록으로 정리해줘. 형식: 1) 고객 요구사항 2) 우리측 합의사항 3) 리스크/확인 필요 4) 후속 액션(담당/기한). 메모: {{meeting_text}}
예시 3. 1:1 면담 기록
아래 1:1 면담 내용을 매니저 기록용 회의노트로 정리해줘. 형식: 1) 현황 2) 이슈 3) 지원 필요사항 4) 다음 점검 시점. 민감한 표현은 중립적으로 정리. 내용: {{meeting_text}}
예시 4. 긴급 이슈 대응 회의
아래 긴급 이슈 대응 회의 메모를 상황실 공유용으로 정리해줘. 형식: 1) 현재 상황 2) 원인 가설 3) 즉시 조치 4) 추가 조치(담당/ETA) 5) 고객 커뮤니케이션 포인트. 메모: {{meeting_text}}
예시 5. 프로젝트 킥오프
아래 프로젝트 킥오프 회의 내용을 실행 중심 회의록으로 정리해줘. 형식: 1) 목표/범위 2) 역할 분담 3) 일정 마일스톤 4) 오픈 이슈 5) 다음 체크포인트. 내용: {{meeting_text}}
예시 6. 스프린트 회고
아래 스프린트 회고 내용을 정리해줘. 형식: 1) 잘한 점 2) 아쉬운 점 3) 다음 스프린트 개선 액션(담당/우선순위/마감). 항목별로 짧게 bullet로 작성. 내용: {{meeting_text}}
예시 7. 의사결정 회의(승인/보류)
아래 의사결정 회의 메모를 경영진 공유용으로 정리해줘. 형식: 1) 논의 안건 2) 결정(승인/보류/반려) 3) 결정 근거 4) 후속 실행 항목 5) 재검토 일정. 메모: {{meeting_text}}

회의록 정리 AI

회의록 정리 AI를 찾는 사람은 보통 “말이 많고 시간이 없고, 액션 아이템만 뽑고 싶다”는 문제를 안고 들어옵니다. 이 페이지는 그 문제를 가장 빠르게 해결하는 프롬프트부터 보여줍니다.

이럴 때 쓰세요
  • 회의 내용을 빠르게 정리할 때
  • 발언자별 핵심 내용이 필요할 때
  • 액션 아이템만 뽑고 싶을 때

예시 입력 / 출력

어떻게 넣고 어떤 결과를 기대하면 되는지 한 번에 보이게 구성했습니다.

예시 입력
신규 랜딩페이지 오픈 일정을 논의했다. 디자이너는 수요일까지 시안을 완료하고, 개발자는 금요일까지 반영하기로 했다. 마케팅팀은 다음 주 월요일부터 광고를 시작한다.
예시 출력
- 회의 목적: 신규 랜딩페이지 오픈 일정 조율 - 핵심 논의: 디자인 일정, 개발 반영 일정, 광고 시작 시점 확인 - 결정 사항: 수요일 시안 완료, 금요일 개발 반영 완료, 다음 주 월요일 광고 시작 - 액션 아이템: 디자이너-시안 완료 / 개발자-반영 / 마케팅팀-광고 집행 - 다음 일정: 다음 주 월요일 광고 시작
