메일 답장 예시

메일 답장 예시를 찾는 사람은 보통 “예의는 지키고 싶은데 문장이 안 떠오른다”는 상태입니다. 이 페이지는 정중한 답장을 빠르게 만드는 데 집중합니다.

이럴 때 쓰세요
  • 정중한 답장이 필요할 때
  • 상사/고객 메일 답장을 써야 할 때
  • 거절/보류 메일을 부드럽게 쓰고 싶을 때

바로 복붙하는 프롬프트

기본/짧은/정중한 버전을 같이 두어 상황에 맞게 바로 써볼 수 있게 했습니다.

기본 버전
아래 상황에 맞는 정중한 업무 메일 답장을 작성해줘. 톤은 예의 바르고 간결하게, 너무 딱딱하지 않게 써줘. 상황: {{situation}} 상대: {{recipient}} 원하는 핵심 메시지: {{message}}
짧은 버전
아래 상황에 맞는 짧고 정중한 메일 답장을 써줘. 상황: {{situation}} 핵심 메시지: {{message}}
정중한 버전
아래 상황에 맞는 매우 정중한 업무 메일 답장을 작성해줘. 서두 인사, 본문, 마무리 인사까지 포함해줘. 상황: {{situation}} 상대: {{recipient}} 핵심 메시지: {{message}}

예시 입력 / 출력

어떻게 넣고 어떤 결과를 기대하면 되는지 한 번에 보이게 구성했습니다.

예시 입력
상대가 일정 변경을 요청했고, 나는 금요일 오후 3시로 변경 가능하다고 답해야 한다.
예시 출력
안녕하세요. 요청 주신 일정 변경 관련하여 확인드렸습니다. 금요일 오후 3시로 조정 가능합니다. 해당 일정으로 진행하겠습니다. 감사합니다.
