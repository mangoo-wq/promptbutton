# 프롬버튼 PromptButton

AI에 뭐라고 써야 할지 막힐 때 쓰는 한국어 도구입니다. 회의 메모, 메일 상황, 고객 문의처럼 대충 적힌 내용을 넣고, ChatGPT·Claude·Gemini에 붙여넣을 문장을 바로 복사할 수 있습니다.

- 사이트: https://promptbutton.app/
- 형태: 정적 웹사이트 / GitHub Pages
- 주요 기능: 업무 고르기 → 내용 넣기 → 붙여넣을 문장 복사
- 현재 제공: 회의록, 메일 답장, 고객 문의 답변, 생활 계산기

## 이런 분께 맞습니다

- AI를 쓰고 싶은데 첫 문장을 어떻게 써야 할지 막히는 분
- 회의록, 메일, 고객 답변을 빠르게 정리하고 싶은 분
- 긴 프롬프트 설명보다 바로 복사해서 써보는 흐름이 편한 분

## 운영 메모

이 저장소는 공개 배포용 정적 파일만 담고 있습니다. 사이트 소스와 운영 문서는 별도 비공개 저장소에서 관리합니다.
